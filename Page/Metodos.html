Metodo: Multiplicadores de Lagrange

El modelo de Lagrange permite maximizar o minimizar una función de dos variables con una restricción,
y puede ser aplicado a dos situaciones:
la búsqueda del equilibrio del consumidor dada una restricción presupuestaria
y la minimización de costos para un nivel determinado de producción de una empresa.

El método de los multiplicadores de Lagrange es una poderosa herramienta
para resolver esta clase de problemas sin la necesidad de resolver explicitamente las condiciones y usarlas para eliminiar variables extra
